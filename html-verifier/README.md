#### Descrição do arquivo

- O arquivo html deve começar com <!DOCTYPE html>
- Cada linha posterior deve conter apenas um item dos abaixo
    - Abertura de tag
        - Indicado por '<'
        - A abertura de tag não pode conter propriedades
    - Fechamento de tag
        - Indicado por '</'
    - Conteúdo
        - Qualquer outra coisa
